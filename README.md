# infinite Realms
Nicholas La Monaco

## Description:

infinite Realms is a full-stack, mobile-first, progressive web application that uses Node.js, Express, MongoDB, and Mongoose with the Scryfall API to 
create a lazy-loading image gallery experience. 

## Deployed Site:
